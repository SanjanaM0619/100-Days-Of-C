
<h1>WHAT IS PROGRAM</h1>
A program is simply a list of instructions written by a programming language that tells the computer what to do 
<br>
<h1>PROGRAMMING LANGUAGE</h1>
A programming language is a way for humans to communicate with computers
Ex:C,Python,Java
<br>
<h1>TYPES OF PROGRAMMING LANGUAGE</h1>
1)Low Level language:<br>
* Directly talk to computer <br>
*These languages are hard for humans to understand but efficient for compuuters
Ex:Machine language - Uses only 0's and 1's like 101010001
Assembly Language -Uses short codes like ADD A,B instead of 0 and 1
<br>
2)High Level languages:<br>
*Easier for humans to understand
Ex:Java,Python
<br>
<h1>C PROGRAMMING</h1>
* C is a general purpose mid level programming language developed by <b>Dennis Ritchie</b> at <b>Bell Laboratories</b> in 1972
<br>
* C is considered as the mother of all programming languages like java,PHP and JavaScript have borrowed syntax/features directly or indirectly from C
<br>
* C is strongly associated with UNIX as it was developed to write the UNIX operating system
<br>
